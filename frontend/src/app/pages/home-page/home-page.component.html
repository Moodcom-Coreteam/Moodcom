L'outil d'analyse Moodcom vous permet d'analyser l'impact de vidéos Youtube sur le public basé sur les sentiments grâce aux commentaires laissés sous cette dernière. L'analyse est réalisée sur les 1000 top commentaires (commentaires les plus aimés).

Comment l'utiliser ?

Rien de plus simple !
Copiez ou recherchez l'URL de la vidéo ou les vidéos que vous souhaitez analyser dans la barre dédiée et ajoutez là. Puis vous n'avez plus qu'à appuyez sur le bouton "Analysez" et le tour est joué !

Sur la page analyse, vous avez accès à 3 types d'analyses différentes :

  • Analyse comparative : les résultats des analyses de chaque vidéo sont présentées sur un seul graphe afin de permettre une comparaison facile des résultats
  • Analyse individuelle : chaque vidéo ajouté dans la liste est analysé individuellement, indépendamment les unes des autres
  • Analyse évolutive : à chaque date de publication des vidéos analysées, on observe les résultats de chaque sentiment (plus d'informations dans l'onglet correspondant)

Une option est disponible !
Vous avez la possibilité de vous connecter avec votre compte Google. Avec cette connexion, vous aurez accès à l'historique de vos analyses. Vous aurez alors le choix entre :

  • Refaire l'analyse : l'analyse est rejouée à la date effectuée
  • Refaire et comparer l'analyse : on affiche les résulats de l'analyse à la date effectuée la première fois et à celle d'aujourd'hui afin de comparer l'évolution des sentiments
